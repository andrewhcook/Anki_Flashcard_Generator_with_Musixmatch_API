This project is a Rust-based API-parser used to generate Anki flashcards to aid the study of a target language via musical lyrical translation.

The folder titled, "Anki Flashcard Deck Generation Tooling" contains two directories that, when built, will yield the pre-release version featured here on Github.

The folder titled, "MusixMatch Anki Flashcard Generator" features a sample directory structure including the necessary file titles to allow for the succesful processing of song lyrics.

This project was built with a Windows 11 target in mind and as such, is entirely untested for MacOS and Linux targets.

Lastly, in order to use this application, you will need a Musixmatch API key (link here, accurate as of 5-4-26: https://www.musixmatch.com/pro/api/pricing)
